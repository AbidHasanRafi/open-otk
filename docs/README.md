## Examples

Check the `/examples` directory in the project root for:

- `simple_chat.py` - Basic chat implementation
- `streaming_chat.py` - Real-time streaming responses
- `chat_session.py` - Session management
- `model_manager.py` - Model management examples
- `model_comparison.py` - Compare different models
- `embeddings.py` - Working with embeddings
